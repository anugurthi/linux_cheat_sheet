# 7-Day Linux for DevOps Roadmap

Goal: In one week gain practical Linux skills needed for junior DevOps work (automation, services, logs, packaging, networking, containers). Deep admin tasks (RAID, SELinux tuning, kernel builds) are intentionally out of scope.
For optional daily deep dives, see External Learning References section in `README.md` and pick ONE related article per day (capture 3 actionable notes).
